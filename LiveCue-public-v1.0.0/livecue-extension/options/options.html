Start local ASR helper

Requires Node.js 20+. Start the relay once, then keep that terminal window open.

  1. Open the downloaded LiveCue extension folder.
  2. macOS: double-click Start LiveCue ASR Relay.command.
  3. Windows: double-click Start LiveCue ASR Relay.bat.
  4. If macOS blocks it, copy the fallback command, paste it into Terminal, choose the command file, and press Enter.
RELAY_SCRIPT="$(osascript -e 'POSIX path of (choose file with prompt "Choose Start LiveCue ASR Relay.command")')" && bash "$RELAY_SCRIPT"

When it works, Terminal shows: LiveCue ASR relay listening on http://127.0.0.1:17395/asr
